# 超星作业 AI 助手

一个用于超星学习通作业页面的油猴脚本，提供题目导出、AI 回填和 UEditor 粘贴限制解除等能力。

## 功能

- 自动解除超星编辑器的 `beforepaste` 粘贴限制
- 在页面左下角显示一个浮动入口，打开后可以导出或回填
- 浮动入口支持拖动，并会记住你上次放置的位置
- 浮动入口有更明显的液态呼吸感动画
- 面板支持开合动画，并拆成「基本操作 / 输出 / 设置」三个板块
- 面板内提供可调参数：浮球大小、浮动速度、面板宽度、透明度、默认打开页、记住位置、减少动效
- 首次会弹出独立的「配置向导」面板，也可以从设置里再次打开
- 一键把当前题目整理成提示词并复制到剪贴板
- 一键读取剪贴板中的 AI 返回结果并回填到页面
- 支持在日志框里展示 `analysis` 解析内容
- 兼容单选、多选、填空等常见题型

## 适用页面

- `https://mooc1.chaoxing.com/mooc-ans/mooc2/work/view*`
- `https://mooc1.chaoxing.com/mooc-ans/mooc2/work/dowork*`

脚本头部当前显式匹配作业查看页和答题页。如果后续超星改了页面地址或结构，可能需要继续调整 `@match` 和选择器。

## 安装

1. 安装浏览器扩展：
   - Tampermonkey
   - 或其他兼容的用户脚本管理器
2. 打开脚本文件 `超星作业 助手.user.js`
3. 通过脚本管理器导入并启用
4. 刷新超星作业页面后使用

## 使用方法

### 1. 导出题目

点击页面左下角的 `AI 助手` 按钮，再点击：

- `1. 导出题目 (含解析要求)`

脚本会抓取页面内 `.questionLi` 题目节点，读取题目标题和选项，拼成一段提示词并复制到剪贴板。

导出的提示词会要求 AI 返回一个 JSON 数组，并且每个条目都包含：

- `questionId`
- `answer`
- `analysis`

### 2. 一键回填

把 AI 生成的结果复制到剪贴板，再点击：

- `2. 一键回填 (同步解析)`

脚本会从剪贴板读取文本，截取其中第一个 `[` 到最后一个 `]` 之间的内容并尝试解析为 JSON 数组。

回填规则：

- 填空题：按 `;`、`；` 或换行拆分多个答案，依次写入对应编辑器
- 选择题：根据答案中的字母匹配页面选项
- 多选题：如果页面暴露了 `addMultipleChoice`，会优先调用它
- 单选题：如果页面暴露了 `addChoice`，会优先调用它
- 其他情况：回退为直接点击对应选项

如果返回数据里有 `analysis` 字段，脚本会把解析内容显示到日志框中。

## 返回格式示例

```json
[
  {
    "questionId": "123456",
    "answer": "A",
    "analysis": "根据题干关键词可知选 A"
  },
  {
    "questionId": "123457",
    "answer": "AB",
    "analysis": "多选题，两个选项都符合"
  },
  {
    "questionId": "123458",
    "answer": "第一空；第二空",
    "analysis": "填空题按分号拆分"
  }
]
```

## 页面依赖

脚本当前依赖以下页面结构或全局对象：

- 题目容器：`.questionLi`
- 题目标题：`h3`
- 选项节点：`.answerBg`
- 其他兼容选项：`.answer_item`、`.options li`
- 填空编辑器：`UE.getEditor(...)`

如果超星页面结构变化，这些选择器可能需要同步更新。

## 注意事项

- 需要浏览器允许脚本读取剪贴板
- 回填前请确认剪贴板内容是合法 JSON 或包含合法 JSON 的文本
- 该脚本依赖当前页面 DOM，不保证适配所有超星旧版/新版页面
- 请遵守学校和平台的使用规范
