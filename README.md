# 超星作业 AI 助手

一个用于 `chaoxing.com` 的油猴脚本，提供题目导出、AI 回填和 UEditor 粘贴限制解除等能力。

## 功能

- 自动解除超星编辑器的粘贴限制
- 一键导出题目到剪贴板，方便交给 AI 处理
- 支持将 AI 返回的答案 JSON 一键回填到页面
- 支持在回填时展示 `analysis` 解析内容
- 兼容单选、多选、填空等常见题型

## 安装

1. 安装浏览器扩展：
   - Tampermonkey
   - 或其他兼容的用户脚本管理器
2. 打开脚本文件 `超星作业 助手.user.js`
3. 使用脚本管理器导入并启用
4. 刷新超星作业页面后使用

## 使用方法

### 1. 导出题目

点击页面左下角的 `AI 助手` 按钮，然后点击：

- `1. 导出题目 (含解析要求)`

脚本会把题目整理成一段提示词并复制到剪贴板。你可以将其发送给 AI，让 AI 按约定格式返回 JSON。

### 2. 一键回填

AI 返回结果后，复制整段 JSON，再点击：

- `2. 一键回填 (同步解析)`

脚本会读取剪贴板内容，自动识别答案并回填到页面，同时把 `analysis` 字段显示到日志框中。

## 返回格式示例

```json
[
  {
    "questionId": "123456",
    "answer": "A",
    "analysis": "根据题干关键词可知选 A"
  },
  {
    "questionId": "123457",
    "answer": "AB",
    "analysis": "多选题，两个选项都符合"
  },
  {
    "questionId": "123458",
    "answer": "第一空；第二空",
    "analysis": "填空题按分号拆分"
  }
]
```

## 兼容性

- 页面域名：`*.chaoxing.com`
- 运行时机：`document-end`
- 依赖页面内的超星题目结构与 UEditor 编辑器

## 注意事项

- 请确保浏览器已授予脚本访问剪贴板的权限
- 回填效果依赖页面实际 DOM 结构，若超星改版，可能需要更新选择器
- 请遵守学校和平台的使用规范
